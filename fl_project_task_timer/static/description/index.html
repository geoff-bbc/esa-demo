Project Task Timer

This module allow to track time and add timesheet in project automatically. It's allow multiple user to track timesheet.

Task Kanban View

In Kanban view, add start and stop timer button.

Users can easily start and stop timer of a Task.

Task Timesheet

In Form view, add start and stop timer button.

The user can easily log timesheet using a timer. It allows logging timesheet for multiple users.

When clicking on the Start button, then a timer will start and log timesheet with start DateTime data.

After Click on Stop Button timer stop and add timesheet with end DateTime data and time duration.

Task Timesheet

In Form view, add start and stop timer button.

The user can easily log timesheet using a timer. It allows logging timesheet for multiple users.

When clicking on the Start button, then a timer will start and log timesheet with start DateTime data.

After Click on Stop Button timer stop and add timesheet with end DateTime data and time duration.

Note: This module is tested and working in Odoo Community Edition but not tested in Enterprise Edition.
